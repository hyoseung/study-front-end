# 미디어 쿼리

## 미디어 쿼리란?

* 미디어에게 질문하고 감지하여 웹사이트를 변경하는 기술
* 컴퓨터나 기기의 환경 또는 종류를 감지해야 그 기기에 맞게 웹 사이트의 구조를 바꿀수 있음
